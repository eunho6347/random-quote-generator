# Random Quote Generator

버튼을 누를 때마다 **랜덤 문구**를 보여주는 간단한 웹앱입니다.

- 데모: https://eunho6347.github.io/random-quote-generator/

## 기능
- 버튼 클릭 시 문구 무작위 출력
- 같은 문구 연속 방지
- 간단한 페이드 애니메이션

## 기술
- HTML, CSS, JavaScript(바닐라)

## 실행 방법
1) 이 저장소를 클론하거나 ZIP으로 다운로드  
2) `index.html`을 브라우저로 열면 바로 실행됩니다.
